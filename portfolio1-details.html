Smartphone Inventory Management System

This project involved developing a Python-based smartphone inventory management system with full CRUD (Create, Read, Update, Delete) functionalities. The system was designed to streamline inventory management for a smartphone retailer, allowing staff to efficiently manage customer orders, track stock, and update inventory records.

Key features include a user-friendly interface for data entry, an automated update system for stock levels, and the ability to delete records for obsolete or sold-out products. The system also integrates with existing customer data systems, improving overall operational efficiency.

Technologies Used

  • Python

Key Features

  • Full CRUD functionalities
  • Seamless integration with customer data management systems
  • User-friendly interface for inventory tracking and updates
