Turn-Based Role-Playing Game (RPG) Simulation

Developed a Python-based Turn-Based Role-Playing Game (RPG) simulation, implementing core mechanics such as a combat system where players and enemies alternate actions. The project featured a dynamic damage calculation algorithm to ensure balanced gameplay, creating an engaging player experience.

The game allows players to select characters with different abilities, enhancing strategic gameplay. A detailed game design document was created, outlining all mechanics, abilities, and combat interactions.

Technologies Used

  • Python

Key Features

  • Turn-based combat system with strategic decision-making
  • Damage calculation algorithm for balanced gameplay
  • Detailed game design document with mechanics and abilities
