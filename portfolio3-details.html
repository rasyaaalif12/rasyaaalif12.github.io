Mobile Game Data Analysis

Conducted data analysis to identify key profitability factors in mobile video games, aiming to maximize revenue with minimal development effort. Leveraged Python (Pandas, Matplotlib) for statistical analysis and data visualization, delivering insights that helped optimize game design and monetization strategies.

By analyzing in-game purchasing behaviors, user retention, and engagement data, the analysis provided a roadmap for improving player retention and boosting in-game purchases. The results informed strategic decisions for game development and marketing.

Technologies Used

  • Python (Pandas, Matplotlib)
  • Jupyter Notebooks

Key Features

  • In-depth analysis of user behavior and purchasing patterns
  • Optimized monetization strategies for mobile games
  • Data visualization using Python and Tableau
